ES5数组新增方法

2017.07.13 Emon0526  热度 ℃

早在很久之前就看到有人提出lodash underscore 这类的lib可能会被淘汰掉
在ES5中，数组新增了以下方法，其参数都是 function 类型：

  • forEach
  • map
  • filter
  • indexOf
  • reduce

forEach 循环

这个方法很像jq中的ecah()

forEach 的 function 回调 有三个参数：
第 1 个是遍历的数组内容
第 2 个是对应的数组索引
第 3 个是数组本身

需要注意的是第 1 个和第 2 个参数正好是相反的。

map 映射

#filter 过滤

返回数组中大于10的数字组成新数组

#indexOf()

和字符串的indexOf方法类似

返回数组的整数索引值，如果没有找到，返回-1

判断一个元素是否在数组里，不在则更新数组
