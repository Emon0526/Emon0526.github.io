css居中

2017.12.08 Emon0526

开发过程中，经常会遇到需要居中（水平居中、垂直居中、水平 + 垂直居中）的情况。

主要有以下几种：

  1. 水平居中：内联元素（inline）
  2. 水平居中：单个块级元素（block）
  3. 水平居中：多个块级元素
  4. 水平居中：多个块级元素（使用 flexbox 布局来解决）
  5. 垂直居中：单行的内联元素
  6. 垂直居中：多行的内联元素
  7. 垂直居中：已知高度的块级元素
  8. 垂直居中：未知高度的块级元素
  9. 水平垂直居中：已知高度和宽度的元素
  10. 水平垂直居中：未知高度和宽度的元素
  11. 水平垂直居中：使用 flexbox 布局来解决
